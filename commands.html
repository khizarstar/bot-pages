Nova core commands

🛡️ Moderation & Safety

/moderation Opens the main core moderation interface to log community punishments, manage user bans, and review server safety layers.
/automod Configures the automatic content filters, anti-spam configurations, and custom blocklists to secure your channels automatically.
/warn Issues a formal security warning to a user, logging it directly into their database moderation profile history.
/unwarn Removes an active warning entry from a specified member, reducing their current punishment tier score.
/warnings Queries the internal storage network to list all chronological server infractions and notes logged against a user account.
/report Provides a secure reporting utility for members to safely flag rules violations directly to a hidden staff review room.
/stickymessages Pins an essential message tightly to the bottom of a specific channel, ensuring it automatically updates over active conversations.

⚙️ Administration & Setup

/setup Launches the master interactive server setup utility to bind core channels, tracking features, and bot logging modules.
/config Opens a secure administrative dashboard to modify background feature switches and edit default module values.
/autosetup Instantly deploys optimized preset server categories, voice configurations, and basic standard moderation channels.
/custom-commands Enables staff teams to design and register specific trigger macros that instantly return custom server response designs.
/announcement Formats professional network broadcast frames with clean layouts to communicate major updates or developer schedules.
/reactionroles Binds automated server identity roles to visual message reaction elements, enabling users to click to choose preferences.

📊 Utility & Statistics

/userinfo Fetches precise individual account info, including exact platform entry dates, snowflake IDs, and active server roles.
/profile Generates a customized user profile showcase card with tracking milestones, level standings, and social status tags.
/bot-profile Displays the public utility identification parameters, authorization credentials, and profile specifications of NovaCore.
/serverstats Compiles detailed numeric dashboard updates outlining total account numbers, online member counts, and general server health.
/embed Opens an interactive visual builder to construct custom message arrays, design layout borders, and post professional text cards.
/tickets Deploys an independent, secure customer service panel where members can instantiate hidden rooms to get help from team moderators.
/suggestions Creates a designated feedback collection hub allowing server members to enter ideas and submit real-time structural votes.
/search Runs automated background searches across systemic database lines to extract explicit knowledge bases without opening a web browser.
/tools Opens an all-in-one menu filled with operational formatting guides, server checking scripts, and standard analytical units.

📈 Metrics & Engagement

/levels Compiles the formal server activity hierarchy, grading total message counts into structured score boards.
/giveaway Launches a standalone setup wizard to organize prize giveaways, manage operational countdown durations, and select drawing winners.
/message Checks text data entries to report on specific individual message counts within standard community chats.
/messages Displays the complete server analytics log tracking the overall message metrics across every active text channel.
/invite Creates safe, optimized connection links tracking back to specific invite generation modules.
/invites Audits the database system to confirm total registration conversions driven by specific staff or user invite references.

🎮 Fun & Entertainment

/fun Displays a diverse catalog of casual textual scripts, humorous response phrases, and chat mini-games to break the ice.
/games Launches dynamic chat mini-games complete with automated logic scoring tracking built right into your channel.
/casino Allows users to test their luck in simulated gaming modules using their virtual server currency balances.
/economy Brings up the virtual banking interface where users check wallet assets, evaluate marketplace parameters, and wire funds.
/images Applies creative graphical layers, photo mutations, and meme templates to specific user profile photos instantly.
/activities Integrates dedicated platform game applications directly inside an active Discord voice channel environment.
/family Simulates fun relationship bonds, marriages, and user connections within your server's casual meta-community.
/birthdays Logs and keeps track of user birthday milestones to automate celebratory messages when their date rolls around.
/music Gives users control over high-fidelity audio streams, track requests, and queue systems in voice channels.
/radio Hooks live broadcast internet radio signals into any voice room for continuous background music.
/afk Flags a user profile as away, instantly notifying any members who mention them with an automated notice.
/notepad Provides a small, secure text workspace where users can save personal notes, links, or text logs within the bot.
/thanks Sends formal appreciation notifications to helpful users, tracking total community gratitude tokens.
/voice Allows members to easily adjust parameters, user caps, and permissions on their temporary custom voice lobbies.

ℹ️ Bot System Info

/help Displays an overview of all accessible module groupings along with diagnostic details on how to inspect specific options.
/bot Provides an absolute diagnostic update showing technical ping speeds, hosting runtime statistics, and build updates.
/developers Credits the core engineering team and shows links to official project contributions and support.
/guild Exposes explicit infrastructure credentials, localized settings, and server identity codes for troubleshooting.
