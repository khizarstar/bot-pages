Nova core commands

🛡️ Moderation

/moderation Opens the main administrative command hub dashboard to quickly issue bulk staff punishments, adjust safety limits, or oversee temporary restrictions.
/automod Configure the real-time protection matrix, customize custom Regex keyword blocks, and adjust sensitivity parameters for text filters.
/warn Logs a formal rule infraction to a targeted user's profile and triggers escalation sequences if past offenses are discovered.
/unwarn Removes a specific infraction point or caution record from a member's moderation file, lowering their active punishment escalation tier.
/warnings Queries the internal database to fetch a clean chronological history of all formal warnings logged against a member.
/report Allows community members to securely flag suspicious activity or malicious users directly to hidden staff review rooms.
/stickymessages Pins a persistent information embed to the bottom of a fast-moving chat channel, automatically reposting it to guarantee high visibility.

⚙️ Core & Setup

/setup Launches an interactive guidance menu to initialize essential channels, background systems, and root server tracking links.
/config Provides deep structural tools to toggle optional modules, tweak feature criteria, and customize default bot preferences.
/autosetup Instantly deploys a pre-configured template of standard community channels, category structures, and typical permission layers.
/help Displays an overview of all accessible module groupings along with diagnostic details on how to inspect specific options.
/bot Displays core details regarding the application's hardware metrics, library files, and general performance speeds.
/developers Credits the core development team and showcases official engineering contributions behind the application framework.

📋 Utility & Analytics

/userinfo Pulls comprehensive account history metrics including exact registration timestamps, unique snowflake IDs, and active server roles.
/profile Renders an engaging profile card showcasing a user's customized text description, accumulated accolades, and standing status.
/bot-profile Displays the public identity bio, configuration properties, and authorization structures tied to the bot profile.
/serverstats Compiles dynamic data panels showcasing real-time member ratios, bot percentages, and general chat milestones.
/embed Opens an interactive visual builder to construct, preview, and output professional informational embeds with tailored borders.
/tickets Initializes automated support centers where members can claim isolated chat spaces to safely resolve technical problems with your staff.
/suggestions Creates interactive community suggestion frames where users submit feedback and cast votes using built-in system buttons.
/search Performs swift metadata queries across indexed documentation references without forcing managers to leave the platform container.
/tools Houses an assortment of convenient server tracking metrics, calculation aids, and format evaluation parameters.
