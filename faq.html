Knowledge Base & Common Inquiries

01. How do I invite NovaCore to my Discord server?
You can add the bot by clicking the official Invite Bot link on our homepage. Ensure you have "Manage Server" or "Administrator" permissions on the target server. For full operational capability, we recommend keeping the default requested permission flags checked during authorization.
02. Why are the slash commands not displaying in my server?
If / commands are missing, it is typically because the bot was invited without the proper application command permissions. To fix this, open your Server Settings > Integrations > NovaCore, and verify that the bot has explicit access to use slash commands. If issues persist, re-authorize the bot using our official invite link.
03. How do I change the default settings or logging channels?
Execute the /setup command to begin configuring your system variables. This utility launches an interactive layout panel where you can route infraction notifications, welcome messages, join/leave tracking updates, and mod logs to your preferred isolated text channels.
04. Does the AutoMod system protect against malicious links and media?
Yes. NovaCore uses advanced internal filters to monitor, catch, and strip phishing targets, unauthorized links, and spam behavior. You can configure precise blocking layers, create safe white-lists, and set up automatic moderation levels by triggering the /automod administrative command dashboard.
05. How does the built-in warning system keep track of infractions?
When a moderator executes /warn against an account, NovaCore commits that incident report securely to its internal cloud infrastructure. These records persist even if the user leaves the server. Staff can pull the complete chronological violation log at any time using /warnings.
06. Can I construct customized announcement blocks and embed frameworks?
Absolutely. Use the /embed command wizard to design professional layouts without messing with raw JSON strings. The tool allows you to specify customized hex-color borders, insert custom image links, structure footers, and format system descriptions into high-quality textual cards.
07. How do I establish clickable user reaction roles?
Run the /reactionroles setup module. The engine will guide you through assigning roles to clear message reaction triggers. Ensure that the integrated role belonging to NovaCore sits higher in your server's role list than the roles it is trying to assign, or it will run into an access error.
08. How does the server level and XP tracking network function?
NovaCore logs active conversation metrics across your standard chat channels. Members earn arbitrary experience points (XP) dynamically as they converse. Server owners can customize rewards, configure multipliers, track progression standings via /levels, or block certain channels from giving XP entirely.
09. Are customer support tickets isolated and secure from public eyes?
Yes. When an individual initializes a panel room via the /tickets component, NovaCore instantiates a secure, hidden channel visible only to that specific user and your server's designated staff. When a ticket is completed, moderators can save or clear the entire conversation transcript.
10. Can NovaCore stream audio content directly into my voice channels?
Yes. By employing our high-fidelity entertainment layer via /music or /radio, users can pipe streaming audio networks directly into active voice rooms. The engine handles queue management, play parameters, volume overrides, and loop scripts.
11. Is user profile data shared across multiple discord servers?
No. While global data configurations track bot-wide statistics, specific user profile milestones, server warnings, economy structures, and level metrics remain tightly isolated to your individual server's unique network identification parameters to protect user privacy.
12. What should I do if the bot stops responding to inputs?
First, query the system status directly by running /bot to check live latency speeds and hosting metrics. If the bot is completely non-responsive, verify that it has permission to view the channel and read text history. If a global outage occurs, technical updates are posted directly to our Support Server.
13. Can we create custom text macro triggers for our community?
Yes. If you need repetitive messaging shortcuts or automated responses for common community needs, you can register and save explicit macro structures through the /custom-commands control module.
14. How do I change or adjust user currency allocations in the economy engine?
Administrators can manage virtual server balances directly through the master /economy console. This control layer enables staff to adjust currency values, establish marketplace parameters, reset local leaderboards, or audit user transactional data logs.
15. Where do I submit feature suggestions or bug disclosures?
You can route ideas directly through the interactive /suggestions block inside your server, or use the /developers credentials link to access our dedicated system infrastructure, where you can speak to our engineers and open code tickets.
