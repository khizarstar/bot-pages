Privacy Policy

This Privacy Policy explicitly details what persistent information NOVACORE collects from your Discord guild ecosystem, how that specific data is maintained, and the clear rights you hold as a guild administrator or user.

1. Data We Collect & Scope

To deliver advanced community moderation, automatic logging, leveling systems, and safety features, NOVACORE processes essential data objects received from the Discord API structure. This is strictly limited to:

  • Discord Identifiers: User IDs, Server (Guild) IDs, Channel IDs, and Role IDs. These allow us to link permissions, configurations, and leveling profiles accurately.
  • Message Metadata: Message IDs are temporarily checked solely to execute features like Auto-moderation, anti-spam protections, and log configurations.
  • Configuration State: Custom welcoming logs, localized moderator warning points, and ticket system structures designed by server staff.

What We Do Not Collect: We never collect, cache, or ask for clear-text personal credentials, emails, authentication passwords, token arrays, or financial transaction details.

2. How We Utilize Your Data

The operational data objects processed by the bot backend are reserved exclusively to fuel functionality components. We process information to:

  • Provide automated moderation actions, logging data updates, and verify anti-nuke server triggers.
  • Track localized member XP metrics to distribute engagement activity rewards.
  • Troubleshoot technical faults, trace backend bugs, and mitigate widespread infrastructure abuse or spam networks.

NOVACORE maintains a strict zero-monetization policy over user data. We do not sell, rent, license, or trade your server context or individual data to third-party advertising companies.

3. Security Architecture & Storage

All server rules, configuration layouts, and user points are held securely using an industry-standard MongoDB database configuration. Infrastructure networks are deployed via Opticalink remote computing setups, applying standard firewall defenses to isolate sensitive cache elements.

While our technical layout implements strict safety measures to block malicious intrusion, please be reminded that no digital system or network data stream over the web can guarantee complete invulnerability.

4. Data Retention & Automatic Purging

We believe in maintaining data minimization practices. If NOVACORE leaves a Discord server guild voluntarily or via kick/ban actions, all localized server logs, configuration setups, and data indexes are automatically earmarked for deletion within 30 days.

Exceptions to this automated cleanup window apply only to specific data properties strictly required to maintain global bot safety, anti-abuse tracing, platform blacklist integrity, or to satisfy valid legal enforcement demands.

5. Data Deletion & Privacy Rights

Server owners and standard individual users maintain full authority over their stored records. If you wish to manually initiate an immediate wipe of your user footprint or stored server metrics from our MongoDB system, you can contact our technical center directly.

Please transmit your official request along with your respective Discord snowflake IDs to our email inbox: contact.novacore.bot@gmail.com. Requests are typically audited and permanently processed within 7 business days.
